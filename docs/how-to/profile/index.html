Get Profile

Set Profile Name
PUT /api/{SESSION}/profile/name		✔️✔️✔️
Set Profile Status (About)
PUT /api/{SESSION}/profile/status		✔️✔️✔️
Set/Delete Profile Status Picture
PUT /api/{SESSION}/profile/picture
DELETE /api/{SESSION}/profile/picture		➕➕➕
