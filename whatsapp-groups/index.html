WhatsApp Groups Automation

WAHA provides an intuitive, private-hosted, free HTTP API designed to effortlessly automate WhatsApp Groups management.

Simplify integration processes, automate posting, and parse posts from WhatsApp Groups with ease using HTTP API!

Get Started

How To Use?

1. Start WAHA
Run one command in the terminal on your own laptop or server
2. Pair Number
Connect your or any other number by scanning QR code
3. Use WhatsApp
Send messages via HTTP API
or receive via Webhook

Why WAHA?

It's free!

WAHA Core version is always free, no limits in messages or time!
WAHA Plus version have no licence expiration!

Self-hosted solution

No strings to suspicious SaaS WhatsApp API solutions. Install it on your own server!

HTTP / REST API
