{{< callout context="note" title="Use a real browser name" icon="outline/info-circle" >}}
We recommend changing only the device name and keeping a real browser name: `Chrome`, `Firefox`, `IE`, `Opera`, `Safari`, `Edge`.

On **GOWS**, `Desktop` is supported as well - it shows the device name on its own, with no browser name in front of it.

Any other value makes WhatsApp show a generic "Other device" and drop the device name.
